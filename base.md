1. 配置
git config -- global user.name "Nicholas"
git config -- global user.email "Nicholas@xx.com"


2. 操作
git init 


3. 添加到仓库
git add base.md

git commit -m "本次提交描述"

添加到仓库完成 有一个版本


4. 查看所有版本
git log


5. 查看仓库状态
git status


6. 撤销
git reset


7. 版本回退
git reset --hard 12345


8. git reflog 
所有git 仓库 操作记录
